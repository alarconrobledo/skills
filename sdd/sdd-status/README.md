# sdd-status

## Qué hace
Muestra el estado actual del proyecto SDD de un vistazo: features en progreso, tareas completadas, próximos pasos. Ultra-ligera, solo lee `state.json`.

## Cuándo usarla
- Al retomar una sesión de trabajo
- Para ver qué queda pendiente
- Para orientarse rápidamente en el proyecto

## Cómo invocarla
```
/sdd:status
/sdd:status [feature-name]
```

## Flujo SDD
Puede usarse en cualquier momento del flujo para ver el progreso.

## Origen
Workflow SDD — Specification-Driven Development
